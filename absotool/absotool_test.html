  • 先按F12并切换到Console
  • 按[Z + 1]初始化后，即可拖动除cls-4之外的element。因为.cls-4没有init
  • 选中任意element后，可按[WASD]调整属性。默认调开启了adjustWidthHeight [Z + 2]，也就是调宽高。
  • 你可以[SHIFT + WASD]来同时调整这一个class的多个元素
  • 也可以[CTRL + 左键]来自由选择多个要调整的元素
  • 调整过程中可按[Z + Q]切换步长。你可以在配置中设置，默认为[1,100]
  • 如果你在wasd过程中觉得那些黑底的提示妨碍到你，可按[Z + T]隐藏。调好之后在[Z + T]或者[C + T]打印出来
  • 然后[X + 1]可以show那些display: none的element，你应该可以看见cls-3了。[X + 2]则把它们hide回去
  • [X + 3]则会帮没有底色的element加底色，如果觉得颜色太丑，以后可在配置中设置你喜欢的颜色。[X + 4]取消加上去的底色
  • 允许你动态addSelector和removeSelector。你可以试着在Console中运行这段代码
    absotool.addSelector(".cls-4");
  • 布局好之后可以按[C + 2]打印宽高，[C + 3]打印位置(offset())，[C + 4]打印z-index
  • 如果下标不符合你的要求，可以修改配置widthHeightReverse和topLeftReverse来调整
  • 也可应按[C + 5]，[C + 6]， [C + 7], [C + 8]分别单独打印宽，高，top，left
  • 这些C方法会保存对应的数据到localStorage，所以按[V + 2]、[V + 3]和[V + 4]就可以恢复布局的宽高、位置和z-index
  • 如果觉得快捷键设置得不合理不顺手，可按[Z + H]打印keycode，再把keycode修改到配置里
  • 以上就是大致的功能，配置内容可查看代码
